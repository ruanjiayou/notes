## 副作用
| 类型       |                                              |
| ---------- | -------------------------------------------- |
| 后坐力     | 水平后坐力比垂直麻烦(水平是左右，垂直是向上) |
| 后坐力回复 | 开火后准星飘回来的时间                       |
| 射击稳定性 | 画面摇动                                     |
| 开镜速度   |                                              |
