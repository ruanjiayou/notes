# 动态接口

## 设计

| 字段        | 类型   | 说明     |
|-------------|--------|--------|
| _id         | string | 唯一 id  |
| title       | string | 标题     |
| desc        | string | 描述     |
| content     | string | 业务代码 |
| status      | number | 状态     |
| createdAt   | date   | 创建时间 |
| updatedAt   | date   | 修改时间 |
| publishedAt | date   | 发布时间 |
