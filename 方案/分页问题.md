# 分页

## 常见问题
- 有总页数的需求
- 数据变化造成上下页重叠
- 列表返回的数组长度与每页数量不一致
- 根据请求参数返回格式
  - count,返回总数
  - cursor,根据游标查询
- 缓存
  - 详情根据id缓存对象
  - 列表只缓存id,分页使用游标.需拆分情况如下
    - 单key多于百万数据
    - 读写频繁(秒级多次发布下线不适合)
    - 排序规则复杂(多字段权重)
    - 极热列表(QPS极高)
  - 

```typescript
type Wrapper {
  code: number;
  message: string;
  data: object | {
    list: object[]
    ended?: boolean;
    total?: number;
    prev_cursor?: string;
    next_cursor?: string;
  }
}
```
